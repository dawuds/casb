# _research

Working notes, reading list, scratch. **Not** citation-gated.

Multi-agent workflow output also lands here first — agents will draft capability sketches, vendor capability claims, policy archetypes, regulator mappings into this folder. Each workflow run gets its own dated subfolder:

```
_research/
  YYYY-MM-DD-<wedge>-<lens>/
    raw/        Agent raw outputs
    notes.md    What was promoted, what was rejected, three-lens sign-off
```

Promotion to `01-` … `08-` requires:

1. Citation gate passed (vendor claims corroborated, regulator claims clause-cited, analyst material labelled as opinion).
2. Three-lens review (Architect / Product / Compliance) recorded.
3. Schema compliance for the target folder.

Nothing in `_research/` is considered authoritative.
